# Mesto Frontend

### Технологии при разработке:

- React 18.1 (CRA)
- JavaScript (ES6)
- Flex-box
- Grid layout
- БЭМ (Nested)
- HTML 5 (Semantic)
- ООП

---

### Как запустить frontend

**Установить зависимости**

```sh
npm i
```

**Запустить проект**

```sh
npm run start
```
